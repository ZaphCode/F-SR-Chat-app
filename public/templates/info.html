WHAT IS F-SR?

F-SR is a real-time chat application, developed using many web technologies focused on high performance. I created this app primarily to understand how websockets work and to learn how to combine different frontend and backend tools, improving my full stack skills.

Backend

On the server side, this application is built with the python web framework, FastAPI, and uses Redis Stack as main and cache database. Additionally, F-SR works with a cloud service called Cloudinary to handle file uploads, and also implements websockets for real-time requests.

Frontend

On the other hand, on the client side, I used Jinja2 template engine to create the website structure and Tailwind css to add styles. I also implemented some javascript to make the websockets connections and add extra functionality.
